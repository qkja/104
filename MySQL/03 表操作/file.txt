1. 创建表的时候,我们可以指定校验集,字符集和存储引擎,不同的存储引擎的表在文件中是不一样的
   1.1. Innodb: 是两个文件, MySIM:是三个文件
2. 产看表结构, desc 
3. 查看创建语句, show create table 表民
4. 修改表的时候,我们可以修改字段的名字,删除字段,和添加一个字段,以及修改字段的类型
   4.1 删除一个字段: alter table 表名 drop 字段名
   4.2 添加一个字段: alter table 表明 add 字段名 字段类型 after 字段名
   4.3 修改一个字段类型: alter table 表明 modify 字段名 新的类型
   4.4 修改字段名: alter table 表明 change 字段名 新的字段民 字段类型
   4.5 修改表明: alter table 表明 rename to 新的表明
